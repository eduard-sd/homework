mkdir hello - создание папки hello
cd hello - открываем директорию hello (копирование строки из проводника ctrl+c, shift+insert - вставить в гит баш, поменять слэши на обратные)
touch hello.html - создает файл hello.html

git init - Создайте новый репозиторий


git add hello.html -  добавить в отслеживание файл хелло.  Если вы хотите добавить изменения в репозиторий, используйте команду. 
git add . - («.»). Это самый краткий и удобный путь для добавления всех изменений в файлы текущего каталога и его подкаталоги

git commit -m "First Commit" - добавить комментарий к файлу хелло/ Если вы опустите метку -m из командной строки, git перенесет вас в редактор (выход - ESC, ввести :wq и нажать Enter)


git status - чтобы проверить текущее состояние репозитория.


git сheckout - для отмены изменений.
git reset - Если вы решили, что не хотите коммитить изменения, команда состояния напомнит вам о том, что с помощью команды git reset можно снять индексацию этих изменений. ОТмена git add
